# Crie um arquivo XML

```
<?xml version="1.0"?>
<Configuration>
<Server>
    <Collection>[Nome da Collection]</Collection>
    <url>[URL do TFS]</url>
</Server>
<PAT>
    [TOKEN]
</PAT>
</Configuration>
```

# Exemplo:

```
<?xml version="1.0"?>
<Configuration>
<Server>
    <Collection>DefaultCollection</Collection>
    <url>fabrikamfiber.com/tfs</url>
</Server>
<PAT>
    wfcddxzadnwppjyspzlrapel6wd5qe4ebklxko5rvdu56ge2q
</PAT>
</Configuration>
```
